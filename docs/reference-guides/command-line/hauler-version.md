### Command Overview

* Print Hauler's current version.

```yaml
Usage:
  hauler version [flags]

Aliases:
  version, v

Flags:
  -h, --help   help for version
      --json   toggle output in JSON

Global Flags:
  -l, --log-level string    (default "info")
```
